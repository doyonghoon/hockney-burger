4. [ ] `설탕`: 고기랑 버무릴 때 씀.

# 만들때 필요한 도구
1. 위생장갑
2. 큰 bowl
3. 후라이팬
5. 양파 볶을 때 휘저을 긴 막대기

# 순서
1. 양파+버터를 먼저 존나 볶음. 누리끼리해지면 끔.
2. 큰 양동이 같은 거에 옮기고 차갑게 식힘.
3. 고기랑 이것저것 큰 공간에 쳐넣고 차가운 양파도 넣음.
4. 막 비벼서 섞고 약 160g 씩 손으로 떼어서 둥글게 만듬.
5. 끝.
